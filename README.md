# Retail SQL Analysis  

SQL-based retail grocery database analysis using aggregate functions, joins, filtering, and transaction operations to extract business insights.

---

## Project Overview  

This project analyzes a grocery retail database to answer key business questions related to:

- Employee salary distribution  
- Customer balances by province  
- Customer concentration by city  
- High-volume product sales  
- Customer purchase behavior  
- Sales performance in 2019  

The queries demonstrate practical SQL skills used in business analytics environments.

---

## Skills Demonstrated  

- Aggregate Functions (AVG, SUM, COUNT)  
- GROUP BY & HAVING  
- INNER JOIN operations  
- Filtering with WHERE & BETWEEN  
- DISTINCT usage  
- ORDER BY sorting  
- Transaction Queries (INSERT, UPDATE, DELETE)
